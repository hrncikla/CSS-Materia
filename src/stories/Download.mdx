# Download & Setup

You can get started with My Materia by downloading the source files manually or by cloning the repo.

## 1. Download options

- **Download ZIP**: [Download latest release](https://example.com/materia.zip)  
- **Clone via Git**:

```bash
git clone https://github.com/your-org/my-materia.git
```

Then copy the `/materia` folder (or `scss/materia`) into your project.

---

## 2. Vite SCSS setup (optional)

If you're using Vite, you can auto-import variables and mixins in all components by configuring `vite.config.js`:

```js
// vite.config.js
import { defineConfig } from 'vite';
import path from 'path';

export default defineConfig({
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@use "@/scss/materia/tokens" as *;`
      }
    }
  },
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    }
  }
});
```

This automatically injects your tokens into all `.scss` files, so you don't have to manually import them everywhere.

---

## 3. Tip

You can combine the SCSS setup with JavaScript init for interactivity:

```js
import { initMateria } from "../js/materia.js";
initMateria();
```

Useful for ripple effects, animated inputs, etc.
